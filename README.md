# Job Board Application
A Job Board Application built using React.js that helps users search jobs, view job details, save jobs, and apply for jobs.

##Features
-  View available job listings
- Search Jobs by title
- View detailed job information
- Save favourite jobs
- Apply for jobs
- Responsive user interface

## Tech Stack
- React.js
- JavaScript
- HTML
- CSS
- Vite

## How to Run the Project
1. Clone the repository
git clone
https://github.com/sanchitadasniki-del/job-board-react.git
2. Install dependencies
npm install
3. Start the development server
npm run dev
